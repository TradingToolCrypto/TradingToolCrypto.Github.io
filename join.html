Get Started Today

Download Software

Installation is as easy as downloading a file.zip to your desktop,
unzipping,
and drag only the contents from the MQL5 folder into your MT5 directory.
Restart MetaTrader5.

Allow Webrequests

Open MT5 , Tools Tab -> Options -> Expert Advisors Tab
check box "Allow DLL Imports"
check box "Allow WebRequest for listed URL:"
add the correct URL https://www.coinpayments.net

Attach Payment Bot 2.0

Attach the payment bot by dragging the robot name from the navigator window and dropping it on to any chart.
Select a software package within the Bot's Input Parameters to create an invoice.
Then, pay the invoice to activate the license.
